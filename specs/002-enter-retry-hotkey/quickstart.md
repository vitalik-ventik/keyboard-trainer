# Quickstart: Гаряча клавіша Enter (002-enter-retry-hotkey)

**Date**: 2026-07-18 | **Plan**: [plan.md](./plan.md)

## Передумови

- Реалізована фіча 001 (гра запускається, рівні грабельні).
- Запуск як у quickstart фічі 001: `python -m http.server 8080` →
  `http://localhost:8080`.

## Сценарії валідації

### V7. Enter на екрані поразки (US1)

1. Запустити рівень, програти (пропустити шип) → екран поразки.
2. Натиснути Enter → лунає звук кліку, рівень перезапускається з 0%,
   грає ігрова музика — ідентично кліку по «СПРОБУВАТИ ЩЕ».
3. Програти знову; натиснути Enter на ЦИФРОВІЙ клавіатурі (NumpadEnter) →
   так само перезапуск.
4. Програти знову; ЗАТИСНУТИ Enter на 2–3 секунди → рівень перезапустився
   рівно один раз; після старту гри утримуваний Enter нічого не робить.

✅ Очікування: FR-030, FR-032, FR-035, FR-036; SC-009, SC-010, SC-011.

### V8. Enter на екрані перемоги (US2)

1. Пройти рівень на 100% → екран перемоги (з кнопками «НАСТУПНИЙ РІВЕНЬ»,
   «ЩЕ РАЗ», «ДО МЕНЮ»).
2. Натиснути Enter → перезапускається ПОТОЧНИЙ рівень (не наступний),
   зі звуком кліку та ігровою музикою.

✅ Очікування: FR-031, FR-032; SC-009.

### V9. Enter в інших станах — без дій

1. МЕНЮ: натиснути Enter → нічого не відбувається (демо триває, рівень не
   стартує).
2. ВИБІР РІВНЯ та НАЛАШТУВАННЯ: Enter → нічого.
3. ПІД ЧАС ГРИ: Enter → кубик не стрибає, рівень не перезапускається.
4. Екран «ЗАВАНТАЖЕННЯ...»: Enter → нічого.

✅ Очікування: FR-034; SC-012.

### V10. Блокування дефолтної поведінки та фокус

1. На екрані поразки клікнути мишею «ДО МЕНЮ» → у меню клікнути «СТАРТ»,
   програти. На екрані поразки (кнопка могла лишитися у фокусі від
   попередніх кліків) натиснути Enter → виконується САМЕ «СПРОБУВАТИ ЩЕ»,
   одна дія, без подвійного спрацювання і без активації сфокусованої кнопки.
2. Alt+Enter та Ctrl+Enter на екрані поразки → перезапуск НЕ відбувається.
3. У консолі браузера — жодних помилок протягом усіх сценаріїв.

✅ Очікування: FR-033, FR-036; SC-011.

### V11. Регресія фічі 001

1. Кнопки мишею на екранах завершення працюють як раніше («СПРОБУВАТИ ЩЕ»,
   «ЩЕ РАЗ», «НАСТУПНИЙ РІВЕНЬ», «ДО МЕНЮ»).
2. Літери ЙЦУКЕН у грі, Space/стрілки (без скролу), збереження прогресу —
   без змін (вибірково V2, V5 з quickstart фічі 001).

✅ Очікування: незмінність наявної поведінки.

## Критерій завершення

Сценарії V7–V11 проходять; консоль чиста; Network — лише localhost.
