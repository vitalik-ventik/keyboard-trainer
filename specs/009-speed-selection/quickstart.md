# Quickstart: Вибір швидкості гри (Slow / Normal / Fast)

## Prerequisites

- Сучасний браузер (Chrome/Edge/Firefox на Windows 11)
- Локальний HTTP-сервер для ES6 модулів (start-http.cmd)
- Жодних npm-пакетів або систем збірки

## Validation Scenarios

### 1. Налаштування відображається в меню

**Команда**: Відкрити http://localhost:8080, натиснути кнопку «Налаштування»

**Очікуваний результат**: У модальному вікні нижче рядка «Розмір зони стрибка»
присутній рядок «Швидкість» з трьома кнопками: [SLOW] [NORMAL] [FAST].
Активна кнопка — NORMAL (за замовчуванням).

### 2. Перемикання Slow змінює демо-рівень

**Команда**: Натиснути [SLOW] в налаштуваннях

**Очікуваний результат**:
- Кнопка SLOW підсвічується блакитним неоном, NORMAL і FAST — тьмяні
- Фоновий демо-рівень сповільнюється (помітно на око, ≈25% повільніше)
- Закрити налаштування, відкрити знов — SLOW залишається активною

### 3. Гра з обраною швидкістю

**Команда**: В налаштуваннях обрати FAST, закрити налаштування, обрати
рівень 1, натиснути Enter для старту

**Очікуваний результат**:
- Рівень біжить помітно швидше (≈25% швидше за NORMAL)
- Шипи з'являються частіше, пилки обертаються швидше
- Hit-зони (OK/Perfect) масштабовані пропорційно

### 4. Зміна під час гри не впливає

**Команда**: Під час проходження рівня натиснути Escape → перейти в
SETTINGS → змінити швидкість з NORMAL на SLOW → повернутися

**Очікуваний результат**: Рівень продовжується з початковою швидкістю
(NORMAL). Зміна застосується лише після смерті/перемоги та наступного
старту.

### 5. Збереження після перезавантаження

**Команда**: Обрати SLOW, закрити налаштування, оновити сторінку (F5),
відкрити налаштування

**Очікуваний результат**: Активна кнопка — SLOW, демо-рівень повільний.

### 6. Зворотна сумісність

**Команда**: Видалити ключ `dfp_save_v1` з localStorage через F12 → Storage
(або просто перший запуск), оновити сторінку

**Очікуваний результат**: Налаштування створюються заново з `speed: "normal"`,
активна кнопка NORMAL.
