# Data Model: Ліги, рівні та прогрес гравця

**Feature**: 011-league-level-restructure
**Date**: 2026-07-19

## Entities

### 1. League (Ліга)

Група підрівнів, об'єднаних рівнем складності та тематикою.

| Поле | Тип | Опис | Валідація |
|------|-----|------|-----------|
| `id` | `number` | Порядковий номер ліги (1–5) | 1 ≤ id ≤ 5 |
| `name` | `string` | Назва ліги українською | Не порожній рядок |
| `levels` | `Level[]` | Масив підрівнів ліги | Від 1 до 16 елементів |

**Інваріанти**:
- `id` унікальний для кожної ліги
- `levels[i].id` зростають послідовно в межах ліги (але не обов'язково безперервно між лігами)
- Сумарна кількість рівнів у всіх лігах = 31

### 2. Level (Підрівень)

Окремий ігровий рівень з унікальною конфігурацією.

| Поле | Тип | Опис | Валідація |
|------|-----|------|-----------|
| `id` | `number` | Глобальний ID рівня (1–31) | 1 ≤ id ≤ 31, унікальний |
| `leagueId` | `number` | ID ліги, до якої належить рівень | 1 ≤ leagueId ≤ 5 |
| `idx` | `number` | Локальний номер у лізі (1..N) | 1 ≤ idx ≤ 16 |
| `name` | `string` | Унікальна назва рівня | Не порожній рядок |
| `letters` | `string[]` | Активні літери українського алфавіту | 4 ≤ length ≤ 33, лише укр. літери |
| `speed` | `number` | Базова швидкість прокрутки (px/s) | 100 ≤ speed ≤ 500 |
| `spikeCount` | `number` | Кількість перешкод | 10 ≤ spikeCount ≤ 70 |
| `seed` | `number` | PRNG seed для детермінованої генерації | ціле число |
| `bgTheme` | `string` | Ідентифікатор процедурного фону | одне з ~25 допустимих значень |
| `accentColor` | `string` | Акцентний колір рівня (hex) | Валідний CSS-колір |
| `rhythmGroups` | `boolean` | Чи генерувати перешкоди групами | true/false |

**Інваріанти**:
- `id` унікальний глобально
- `letters` містить лише літери з українського алфавіту ЙЦУКЕН
- `speed`, `spikeCount` відповідають складності ліги

**Стан блокування** (обчислюване, не зберігається):
- `locked: boolean` = `id > progress.unlocked`
- Визначається динамічно відносно прогресу гравця

### 3. PlayerProgress (Прогрес гравця)

Зберігається в `localStorage` під ключем `"dfp_save_v1"`.

```json
{
  "version": 1,
  "progress": {
    "unlocked": 1,
    "levels": {
      "1": { "bestPct": 85, "highScore": 1234 },
      "2": { "bestPct": 0, "highScore": 0 },
      "...": {}
    }
  },
  "settings": {
    "difficulty": "EASY",
    "hitWindow": "normal",
    "speed": "normal"
  }
}
```

| Поле | Тип | Опис | Значення за замовчуванням |
|------|-----|------|--------------------------|
| `version` | `number` | Версія схеми даних | 1 |
| `progress.unlocked` | `number` | Глобальний ID останнього розблокованого рівня | 1 |
| `progress.levels[id].bestPct` | `number` | Найкращий відсоток проходження | 0 |
| `progress.levels[id].highScore` | `number` | Найкращий рахунок (очки) | 0 |
| `settings.difficulty` | `string` | Режим складності | `"EASY"` |
| `settings.hitWindow` | `string` | Розмір зони стрибка | `"normal"` |
| `settings.speed` | `string` | Множник швидкості | `"normal"` |

**Правила валідації**:
- `unlocked` ≥ 1, ≤ 31
- `bestPct` ∈ [0, 100]
- `highScore` ≥ 0
- `difficulty` ∈ {`"EASY"`, `"HARD"`}
- `hitWindow` ∈ {`"normal"`, `"large"`}
- `speed` ∈ {`"slow"`, `"normal"`, `"fast"`}

**Санітизація** (при завантаженні):
- Пошкоджені дані → скидання до значень за замовчуванням
- Відсутні рівні в `progress.levels` → додаються з нульовими значеннями
- Некоректні значення → кліпуються до допустимих діапазонів

### Entity Relationships

```
League (1) ────< (N) Level
   │                  │
   │                  │ unlocked залежить від
   │                  │
   └──── прогресивне ─┘
        розблокування

PlayerProgress.unlocked ──── визначає ────> Level.locked (обчислюване)
PlayerProgress.levels[id] ──── зберігає ────> результати для кожного Level.id
```

### State Transitions

**Розблокування рівня**:
```
Level N completed (100%)
  → if N is last level in league L
      → unlock first level of league L+1 (якщо L < 5)
  → else
      → unlock level N+1 (наступний у тій самій лізі)
```

**Скидання прогресу** (крайовий випадок):
```
localStorage.clear() / пошкоджені дані
  → unlocked = 1
  → усі levels[id] = { bestPct: 0, highScore: 0 }
```
