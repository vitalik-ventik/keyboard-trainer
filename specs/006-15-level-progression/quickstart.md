# Quickstart: Валідація 15-рівневої прогресії

**Created**: 2026-07-18
**Feature**: 006-15-level-progression

## Передумови

- Браузер Chrome/Edge 90+ на Windows 11
- Відкрити `index.html` (локально або через статичний сервер: `npx serve .` для ES6-модулів)
- У директорії `sounds/` мають бути файли: `jump.wav`, `explode.wav`, `victory.wav`, `click.wav`
- У директорії `music/` мають бути файли: `menu.mp3`, `game.mp3`, `gameover.mp3`, `win.mp3`

## Сценарії валідації

### V1: Екран вибору рівнів показує 15 рівнів

1. Запустити гру.
2. Натиснути кнопку «Рівні» у головному меню.
3. **Очікується**: Відображається сітка з 15 карток рівнів. Рівень 1 активний. Рівні 2–15 сірі/заблоковані (іконка замка).

### V2: Проходження Рівня 1 розблоковує Рівень 2

1. На екрані вибору рівнів натиснути Рівень 1.
2. Дочекатися завершення рівня (прогрес-бар → 100%, екран перемоги).
3. Повернутися на екран вибору рівнів.
4. **Очікується**: Рівень 2 активний і доступний для натискання.

### V3: Прогрес зберігається між перезапусками

1. Виконати V2 (пройти Рівень 1).
2. Закрити вкладку браузера.
3. Відкрити `index.html` знову.
4. Натиснути «Рівні».
5. **Очікується**: Рівень 2 активний (прогрес збережено).

### V4: Унікальний фон для кожного рівня

1. Запустити Рівень 1.
2. **Очікується**: Темно-синій градієнт із блакитною сіткою та повільними вертикальними лініями.
3. Вийти в меню (кнопка «Меню»), натиснути «Рівні», запустити Рівень 6.
4. **Очікується**: Чорно-зелений фон із падаючими зеленими відрізками («Матриця»).
5. Повторити для Рівня 15.
6. **Очікується**: Чорний фон із криваво-червоними лініями, пульсація всього екрана.

### V5: Клавіатура — блакитні літери пулу

1. Запустити Рівень 1 (літери `А`, `О`).
2. Подивитися на віртуальну клавіатуру внизу екрана.
3. **Очікується**: Клавіші `А` та `О` підсвічені м'яким блакитним (cyan). Решта — тьмяно-сірі.

### V6: Клавіатура — жовто-зелена цільова літера

1. Під час гри на Рівні 1 спостерігати за клавіатурою при наближенні до шипа.
2. **Очікується**: Літера найближчого шипа підсвічена яскравим жовто-зеленим (lime/yellow). Інші блакитні літери зберігають свій колір.

### V7: Клавіатура — червоний спалах помилки

1. На Рівні 1 натиснути літеру, якої немає в пулі (наприклад, `П`).
2. **Очікується**: Клавіша `П` миттєво спалахує яскраво-червоним.
3. Зачекати ~400 мс.
4. **Очікується**: Червоне підсвічування зникає, клавіша повертається до сірого.
5. Натиснути будь-яку іншу клавішу (навіть правильну `А`).
6. **Очікується**: Якщо червоне ще не згасло — воно негайно зникає.

### V8: Три типи перешкод

1. Запустити будь-який рівень.
2. Спостерігати за перешкодами на трасі.
3. **Очікується**: Зустрічаються трикутні шипи, подвійні шипи (два трикутники поруч) та обертові пили (кола із зубцями). Пили обертаються навколо осі.

### V9: Міграція старого збереження

1. Відкрити консоль розробника (F12), виконати:
   ```js
   localStorage.setItem("dfp_save_v1", JSON.stringify({
     version: 1,
     settings: { difficulty: "EASY" },
     progress: { unlocked: 3, levels: {
       "1": { bestPct: 100, highScore: 150 },
       "2": { bestPct: 85, highScore: 200 },
       "3": { bestPct: 0, highScore: 0 }
     }}
   }));
   ```
2. Оновити сторінку.
3. Натиснути «Рівні».
4. **Очікується**: Рівні 1–3 розблоковані. Рівень 4 — заблокований. Рівні 7–15 — заблоковані. На картці Рівня 1: «Кращий результат: 100%», Рівня 2: «Кращий результат: 85%».

### V10: Продуктивність — 60 FPS на всіх рівнях

1. Відкрити інструменти розробника (F12) → вкладка Performance.
2. Запустити Рівень 15.
3. Записати ~5 секунд ігрового процесу.
4. **Очікується**: Середній час кадру ≤ 16.7ms (60 FPS). Немає пропущених кадрів.

## Примітки

- Якщо аудіофайли відсутні — гра працює без звуку (очікувана поведінка).
- Якщо localStorage заблоковано — гра стартує з Рівня 1, прогрес не зберігається між сесіями.
- Для тестування без проходження всіх рівнів можна встановити `unlocked` вручну через консоль:
  ```js
  // Розблокувати всі 15 рівнів
  const save = JSON.parse(localStorage.getItem("dfp_save_v1"));
  save.progress.unlocked = 15;
  localStorage.setItem("dfp_save_v1", JSON.stringify(save));
  location.reload();
  ```
