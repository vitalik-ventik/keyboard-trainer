# Data Model: Адаптивна дуга стрибка

**Created**: 2026-07-19  
**Feature**: [spec.md](./spec.md)

## Entities

### Player (Кубик)

Ігровий об'єкт, що рухається горизонтально зі швидкістю рівня та вертикально під дією гравітації.

| Поле | Тип | Опис | Зміни в цій фічі |
|------|-----|------|-------------------|
| `x` | number | Горизонтальна позиція (px) | Без змін |
| `y` | number | Вертикальна позиція (px, 0 = земля, від'ємне = у повітрі) | Без змін |
| `vy` | number | Вертикальна швидкість (px/s, додатнє = вгору) | **Змінено**: тепер обчислюється адаптивно за формулою `vy = max(GRAVITY * distance / (2 * effectiveSpeed), MIN_JUMP_VELOCITY)` |
| `onGround` | boolean | Чи кубик на землі | Без змін |
| `rotation` | number | Кут обертання в польоті | Без змін |
| `alive` | boolean | Чи кубик живий | Без змін |
| `trail` | array | Слід (трейл) кубика | Без змін |

**State transitions**:
- `onGround: true` → `jump(distance, perfect)` → `onGround: false`, `vy = computedVy`
- `onGround: false` → `update(dt)` → `y <= 0` → `onGround: true`, `vy = 0`
- `alive: true` → `explode()` → `alive: false`

### Spike (Шип)

Перешкода на трасі з літерою-міткою.

| Поле | Тип | Опис | Зміни в цій фічі |
|------|-----|------|-------------------|
| `x` | number | Горизонтальна позиція (px) | Без змін |
| `letter` | string | Українська літера для натискання | Без змін |
| `state` | string | `ahead` / `cleared` / `hit` | Без змін |
| `type` | string | Тип перешкоди (`spike` / `pillar` / `saw`) | Без змін |
| `rotationAngle` | number | Кут обертання (для saw) | Без змін |

**State transitions** (без змін):
- `ahead` → `cleared`: при успішному стрибку
- `ahead` → `hit`: при колізії з кубиком

### Jump Buffer (Буфер стрибка)

Збережене натискання правильної літери в повітрі.

| Поле | Тип | Опис | Зміни в цій фічі |
|------|-----|------|-------------------|
| `jumpBuffer` | Spike \| null | Посилання на шип, для якого гравець натиснув літеру в повітрі | **Змінено**: `consumeJumpBuffer()` тепер обчислює `distance` на основі `spike.x - player.x` на момент приземлення |

**State transitions**:
- `null` → `Spike`: коли гравець натискає правильну літеру в повітрі
- `Spike` → `null`: при споживанні (`consumeJumpBuffer`) або вибуху (`explode`)

## Constants (Фізичні константи)

| Константа | Значення | Опис | Зміни |
|-----------|----------|------|-------|
| `GRAVITY` | 2600 | Гравітація (px/s²) | Без змін |
| `MIN_JUMP_VELOCITY` | 420 | **Нова** — мінімальна вертикальна швидкість стрибка (px/s) | **Додано** |
| `SAFE_MARGIN` | 25 | **Нова** — посадковий зазор за шипом (px) | **Додано** |
| `CUBE_SIZE` | 42 | Розмір кубика (px) | Без змін |
| `SPIKE_W` | 44 | Ширина шипа (px) | Без змін |
| `SPIKE_H` | 48 | Висота шипа (px) | Без змін |
| ~~`JUMP_VELOCITY`~~ | ~~660~~ | Фіксована швидкість стрибка | **Видалено** |

## Формула обчислення швидкості стрибка

```text
distance = gap + SPIKE_W/2 + SAFE_MARGIN
computedVy = GRAVITY * distance / (2 * effectiveSpeed)
player.vy = max(computedVy, MIN_JUMP_VELOCITY)
```

де:
- `gap` = `spike.x - player.x` на момент виклику `jump()`
- `effectiveSpeed` = швидкість рівня з урахуванням множника (`slow`/`normal`/`fast`)
- `SAFE_MARGIN` = 25 (посадковий зазор)
- `MIN_JUMP_VELOCITY` = 420 (мінімальна гарантована швидкість)

## Точки обчислення distance

| Точка виклику | Де обчислюється `gap` | Контекст |
|---------------|----------------------|----------|
| `handleLetter()` | `spike.x - player.x` у момент натискання клавіші | Кубик на землі, гравець натискає правильну літеру |
| `consumeJumpBuffer()` | `spike.x - player.x` у момент приземлення | Кубик щойно приземлився після попереднього стрибка |
| Демо-блок `update()` | `target.x - player.x` у кадрі демо-логіки | Автоматичний стрибок у демо-режимі |
